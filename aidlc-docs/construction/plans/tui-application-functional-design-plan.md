# UNIT-03: TUI Application 機能設計計画

## 概要

UNIT-03（TUI Application）の機能設計を行う。Textualフレームワークを使用したマルチペインTUIの構築、ユーザーインタラクション（キーバインド、ツリー操作、コメント入力）、API層との連携を設計する。

## 計画ステップ

- [ ] ステップ1：LazyGitLabApp のビジネスロジック設計（レイアウト、キーバインド、ライフサイクル）
- [ ] ステップ2：MRListPanel のビジネスロジック設計（ツリー構造、カテゴリ表示、選択イベント）
- [ ] ステップ3：ContentPanel のビジネスロジック設計（Overview表示、差分表示、行選択）
- [ ] ステップ4：CommentDialog のビジネスロジック設計（タイプ選択、入力、送信）
- [ ] ステップ5：HelpScreen のビジネスロジック設計（キーバインド一覧）
- [ ] ステップ6：ビジネスルール（キーバインドマッピング、エラー表示、状態遷移）
- [ ] ステップ7：フロントエンドコンポーネント設計（Textual CSS、ウィジェット構成）
